To install the Deadline standalone Python API, just copy the Deadline folder from this location to your site-packages folder.

To use the Deadline standalone Python API, Deadline Pulse must be running. See the online documentation for more information on running Deadline Pulse:
http://www.thinkboxsoftware.com/deadline/

The Deadline standalone Python API reference documentation can be downloaded from the Deadline downloads page:
http://www.thinkboxsoftware.com/deadline-downloads/
